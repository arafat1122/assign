03About PassForge

PassForge is a small, self-contained tool for generating strong passwords and understanding why a password is strong or weak. Everything runs client-side in plain HTML, CSS, and JavaScript — nothing is logged, stored, or transmitted anywhere.

Generate tab

Pick a length (6–64 characters) and which character sets to include. Click forge password to generate one using your browser's cryptographically secure random number generator (crypto.getRandomValues), not the weaker Math.random() that many tutorials use. Click copy to copy it to your clipboard.

Analyze tab

Type any password into the field to see a live strength readout. The tool calculates:

  • Entropy (bits) — roughly, how many yes/no guesses an attacker would need on average. Calculated as length × log2(pool size), where pool size is the number of distinct characters available given what character types you used.
  • Estimated crack time — entropy converted into a rough time-to-crack assuming a fast offline attack (10 billion guesses/second on a leaked password hash).
  • Pattern flags — quick checks for common weaknesses like repeated characters, sequential runs (e.g. abcd, 1234), and very short length.

Click show to reveal the characters you've typed if you want to double check them.

Why entropy, not just "rules"?

Many sites enforce rules like "must contain a number and a symbol," but a rule-following password like Passw0rd! is still weak — it's short and predictable. Entropy measures unpredictability directly: longer passwords with a genuinely random mix of characters are exponentially harder to guess than short ones dressed up with a capital letter and an exclamation mark.

PassForge is an educational tool, not a vault. It does not save, sync, or remember any password you generate or type — closing or refreshing the page clears everything.
